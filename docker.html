Why have you used Linux in this course?

Docker has its foundation built on top of Linux concepts. So, even if you're a Windows
user, you'll highly benefit from knowing basic Linux commands. Don't worry! I'll teach

you develop on Windows, you might want to use Linux images to speed up your builds
and deployments.
